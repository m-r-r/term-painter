Everything that can be seen as part of a style. This is the core of this crate. All functions ("style modifier") consume self and return a modified version of the style.

fn paint<T>(&self, obj: T) -> Painted<T>

Wraps the style specified in self and something of arbitrary type into a Painted. When Painted is printed it will print the arbitrary something with the given style. T needs to implement std::fmt::Display or std::fmt::Debug.
